Starlax

Add a quick and lightweight HTML5 Canvas based parallax starfield in the background of your website.

All you need is:

var yourVariable = new Starlax();
Start now

Inline Demos

var starlax = new Starlax({
                            targetCanvas: '#inline-demo',

                            shape:'square',

                            color:'black',

                            size:20,

                            sizeRandom:0,

                            zPos:5,

                            zPosRandom:0
});
var starlax = new Starlax({
                            qtyMultiplier:3,

                            backgroundColor: '#401705',

                            twinkle:false,

                            zPosOpacity:false,

                            color:[
'#D3F8E2','#E4C1F9',
'#F694C1','#EDE7B1',
'#A9DEF9'
]
});

Customize

Syntax: var starlax = new Starlax({ property: value });
Property Default Description
targetCanvas A new <canvas> is created. HTML Canvas element that will be configured. If nothing or a non-canvas element is specified, a background canvas will be created that fills the entire width and height of the window.
qtyMultiplier 1 Multiply the number of stars from the default.
shape 'circle' The shape that stars will take.
fadeIn true Stars will fade into view when the page loads.
fadeInDuration 1 Duration of the fade in. (Value in seconds)
twinkle true Stars will twinkle.
twinkleDuration 1 Duration of each twinkle. (Value in seconds)
backgroundColor null Background Color. Otherwise it's transparent.
color '#ffffff' Color of the stars. Can be an array of color strings for multiple colors.
size 5 Width of stars in pixels.
sizeRandom 0.5 Variety of random values based on size. (Any value from 0 - 1)
zPos 6 The stars position in Z space, and how strong the parallax effect is.
zPosRandom 0.8 Variety of random values based on Z Position. (Any value from 0 - 1)
zPosOpacity true Reduce star opacity according to Z space.
